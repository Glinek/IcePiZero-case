# IcePi Zero Case 
![image](./Images/20260530_203229.jpg)

In this repo there are 4 files, two variants of the top part (one for multicolor printing, one for standard printers) and two wariants of bottom part (one designed for M2 heat inserts and one just for the screews). Whatever you use, you are going to need four M2 screews with length of at least 10mm
